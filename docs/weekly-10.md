# 编程
1. Python获得本月每一天日期
```
def getDayOfMonth():
    myDate = '2022-12-'
    arr = []
    start = 1
    end = 31
    while start <= end :
        if start < 10:
            startStr = "0"+str(start)
        else:
            startStr = str(start)
        newstr = myDate+startStr
        arr.append(newstr)
        start = start + 1
    return arr
```
